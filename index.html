Privacy Policy

本プライバシーポリシーは、スマートフォン向けアプリ「snaptidify」（以下、「本アプリ」）における ユーザー情報の取り扱いについて説明するものです。本アプリをご利用になる前に、必ずお読みください。

This privacy policy explains how user information is handled in the smartphone application "snaptidify" (hereinafter "this App"). Please read this policy carefully before using the App.

1. 収集する情報について / Information We Collect

【デバイス上の写真・メディアデータ】
本アプリは、写真の表示・整理（フォルダへの移動・削除）を実現するために、以下のシステム権限を要求します。 これらのデータは端末内でのみ処理され、外部サーバーや第三者へ送信・収集されることは一切ありません。

  • 写真・画像へのアクセス権限（READ_MEDIA_IMAGES / READ_EXTERNAL_STORAGE）：写真の読み込みと表示に使用します。
  • 写真の部分アクセス権限（READ_MEDIA_VISUAL_USER_SELECTED）：Android 14以降の選択的アクセスに対応するために使用します。
  • メディア位置情報のアクセス権限（ACCESS_MEDIA_LOCATION）：位置情報やGPSデータを収集・取得・送信するために使用するものではありません。Androidの公式仕様（Android Developers公式ドキュメント）により、MANAGE_MEDIAと組み合わせて一括写真整理をユーザー確認ダイアログなしで実行するために必要な技術的要件として宣言しています。本アプリが位置情報・GPSデータを収集・読み取り・利用・送信することは一切ありません。
  • メディア管理権限（MANAGE_MEDIA）：写真1枚ごとにシステム確認ダイアログが表示されることを防ぎ、一括整理をスムーズに行うための特別な権限です。この権限はユーザーが設定画面から明示的に許可した場合にのみ有効となり、強制的に付与されることはありません。

[Device Photos & Media Data]
This App requests the following system permissions to display and organize photos (move to folders, delete). These data are processed only on-device and are NEVER transmitted to external servers or third parties.

  • Photo/image access (READ_MEDIA_IMAGES / READ_EXTERNAL_STORAGE): Used to read and display photos.
  • Partial photo access (READ_MEDIA_VISUAL_USER_SELECTED): For selective access on Android 14+.
  • Media location access (ACCESS_MEDIA_LOCATION): This permission is NOT used to access, read, collect, or transmit any GPS or location data. Per Android's official documentation (Android Developers), this permission must be declared alongside MANAGE_MEDIA to allow batch photo operations (createWriteRequest()/update()) to proceed without showing a per-file confirmation dialog to the user. The App does NOT access, collect, use, or transmit any GPS/EXIF location data whatsoever. Reference: developer.android.com
  • Media management (MANAGE_MEDIA): A special permission that prevents per-photo system confirmation dialogs during batch sorting operations. It is only active when explicitly granted by the user from the in-app settings screen and is never forced.
📌 ストア審査・Data Safety 申告向け補足：
本アプリは ACCESS_MEDIA_LOCATION 権限を宣言していますが、ユーザーの位置情報・GPSデータを収集・利用・送信する目的では一切ありません。 Androidの公式ドキュメント（developer.android.com）が明示するとおり、 MANAGE_MEDIA と組み合わせてユーザー確認ダイアログなしの一括写真整理を実現するために必要な技術的要件として宣言しています。
This App declares ACCESS_MEDIA_LOCATION not to access location data, but because Android's official documentation requires it alongside MANAGE_MEDIA to allow batch photo operations without per-file confirmation dialogs. No GPS/location data is collected, read, or transmitted.
フォルダ情報、自動削除設定、チュートリアル進捗などの設定情報は、端末内のローカルストレージ（SharedPreferences / Room データベース）に保存されます。 これらのデータが外部へ送信されることはありません。

[In-App Settings Data]
Settings such as folder information, auto-delete preferences, and tutorial progress are stored in local storage on your device (SharedPreferences / Room database). This data is never transmitted externally.

2. 広告サービスについて / Advertising Services

【Google AdMob の利用】
本アプリは、広告配信のために Google AdMob（提供：Google LLC）を使用しています。 AdMob は、広告の最適化・配信・不正防止のために以下の情報を収集する場合があります。

  • 広告識別子（Advertising ID / IDFA）
  • IPアドレス（おおよその位置情報の推定に使用）
  • デバイス情報（OS バージョン、機種名、端末またはその他の識別子など）
  • ネットワーク情報（通信キャリア、接続の種類など）
  • アプリの利用状況・インタラクションデータ（広告のクリック・表示状況）

これらのデータの収集・利用については、Google のプライバシーポリシーが適用されます： https://policies.google.com/privacy

広告識別子はユーザーがいつでもリセットまたは広告のパーソナライズを無効化できます。 Android の場合は「設定」→「プライバシー」→「広告」からご確認ください。

[Google AdMob]
This App uses Google AdMob (provided by Google LLC) for advertising. AdMob may collect the following information for ad optimization, delivery, and fraud prevention:

  • Advertising identifiers (Advertising ID / IDFA)
  • IP address (used to estimate approximate location)
  • Device information (OS version, device model, device or other identifiers, etc.)
  • Network information (carrier, connection type, etc.)
  • App usage and interaction data (ad clicks and impressions)

Google's Privacy Policy applies to the collection and use of this data: https://policies.google.com/privacy
You can reset your Advertising ID or opt out of ad personalization at any time via Android Settings → Privacy → Ads.

【User Messaging Platform（UMP）による同意管理】
本アプリは、欧州経済領域（EEA）・英国・スイスにお住まいのユーザーに対して、 Google の User Messaging Platform（UMP）を通じて広告に関する同意を取得しています。 同意は初回起動時に確認され、アプリの「設定」→「広告の同意設定」からいつでも変更できます。 同意が得られない場合、パーソナライズ広告は表示されません。

[Consent Management via User Messaging Platform (UMP)]
For users in the European Economic Area (EEA), UK, and Switzerland, this App collects consent for advertising through Google's User Messaging Platform (UMP). Consent is requested at first launch and can be updated at any time via Settings → "Ad Consent Settings". If consent is not granted, personalized ads will not be shown.

4. データの保持期間 / Data Retention

本アプリがローカルに保存するデータ（フォルダ情報・設定・ゴミ箱の状態管理情報など）は、 アプリをアンインストールするまで端末に保持されます。 アンインストール後はすべての設定データが削除されますが、写真ファイル自体は Android のメディアストアに残ります。

Data stored locally by this App (folder names, settings, trash status, etc.) is retained on your device until the App is uninstalled. After uninstallation, all settings data is deleted, but photo files themselves remain in the Android MediaStore.
